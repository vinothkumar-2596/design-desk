# DesignDesk showcase screenshots

These screenshots power the auto-advancing slider on the featured blog post
(`/brand-guidelines/blog/designdesk-overview`).

The slider expects the following three files in this folder:

| Filename            | What it should show                                         |
| ------------------- | ----------------------------------------------------------- |
| `landing.png`       | DesignDesk landing page (dark — "Design workflows, made efficient") |
| `new-request.png`   | Quick Design request intake form                            |
| `task-detail.png`   | Task detail page (Campaign overview + deliverables)         |

## Capture guidance

- Export at **1600 × 1000 PNG** (or higher) from a 1280-wide browser at 1.25× DPR.
- Crop to the same aspect ratio across the three so the slider feels consistent.
- Keep test data anonymous (no real student / staff names).

Drop the three PNGs into this folder with the filenames above and the slider
will pick them up automatically — no code change required.
